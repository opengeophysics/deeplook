deeplook.optimization.Newton¶

class deeplook.optimization.Newton(initial, precondition=True, maxit=30, tol=1e-05)[source]¶

Minimize a scalar function using Newton’s method.

Given a goal function \(\Gamma(\mathbf{p})\), minimize it iteratively by successively adding steps \(\Delta\mathbf{p}^k\) to a starting estimate \(\mathbf{p}^0\). A step is calculated by solving the linear system:

\[\mathbf{H}(\mathbf{p}^k)\Delta\mathbf{p}^k = -\nabla \Gamma(\mathbf{p}^k)\]

in which \(\mathbf{H}(\mathbf{p}^k)\) is the Hessian matrix and \(\nabla\Gamma(\mathbf{p}^k)\) is the gradient vector of the goal function, both evaluated at iteration k.

The iterations stop when the maximum number of iterations is reached or when the stopping criteria are fulfilled. The default stopping criteria are StopConvergence and StopSmallGradient. You can replace the defaults by assigning a list of stopping criteria to the stopping_criteria attribute.

Parameters:
* initial : 1d array

Initial estimate of the parameter vector.

* precondition : bool

If True, will apply Jacobi preconditioning (recommended).

* maxit : int

The maximum number of iterations.

* tol : float

The tolerance parameter for the stopping criteria. Use the precision of the gradient computations (see equation 9.47 in [Aster2012]).
